# ikipedia

カスすぎるGoogle Chrome拡張機能。
Wikipediaにのみ作用して、ページをとんでもないことにする。

---

## 機能

Wikipediaのページをどうにかしちゃう。


---

## 動作環境

- Google Chrome（Manifest V3対応）
- 対応サイト：*.wikipedia.org のみ

---

## 導入方法

1. このリポジトリをZIPでダウンロードして展開
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」をONにする
4. 「パッケージ化されていない拡張機能を読み込む」をクリックする
5. 展開したフォルダを選択する

---

## 使い方

1. Wikipediaの任意のページを開く
2. ツールバーの拡張機能アイコンをクリックする
3. ツール有効化ボタンを押下
4. おや……？Wikipediaの様子が…。
5. 元に戻すには有効化時と同じボタンを押下

---

## 影響範囲

この拡張機能は以下を保証する。

- Wikipedia（*.wikipedia.org）以外のサイトでは一切動作しない
- ユーザーのデータを読み取り・収集・送信しない
- 外部サーバーへの通信を行わない
- 保存するデータはON/OFFの状態のみ（chrome.storage.local）

---

## 削除方法

1. `chrome://extensions/` を開く
2. 「ikipedia」の「削除」ボタンをクリックする
3. 確認ダイアログで「削除」を押す

削除と同時に、拡張機能が保存していたすべてのデータ（ON/OFF状態）もChromeによって自動的に消去される。
ブラウザやWikipediaには何も残らない。
